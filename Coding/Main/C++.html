Fundamentals of Array Theory

Arrays are very basic data structures, and the questions that examine arrays generally focus on testing the ability to control the code. That is to say, the idea is simple, but it may not be as easy to implement.

Binary Search

Given an ordered (ascending) integer array nums with n elements and a target value target, write a function to search for the target in nums. If the target value exists, return a subscript; otherwise, return -1.

Search Insert Position

Given a sorted array of distinct integers and a target value, return the index if the target is found. If not, return the index where it would be if it were inserted in order.

Find First and Last Position of Element in Sorted Array

After considering the three scenarios, first focus on finding the right interval, then focus on finding the left interval, and make the final judgment based on the left and right intervals.

Remove Element

Given an array nums and a value val, you need to remove all elements with values equal to val in place and return the new length of the removed array.

Remove Zeroes

Given an array nums, write a function to move all 0s to the end of the array while maintaining the relative order of non-zero elements.
